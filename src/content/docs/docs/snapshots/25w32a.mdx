Oprócz miedzianych rzeczy, dodany został nowy "Text Component", `object`.  
Pozwala na wyświetlanie dowolnej grafiki z [Atlasu Tekstur](https://minecraft.wiki/w/Texture_atlas).

Po ludzku, w tekście (na chacie, w lorze itemów, czy nawet na tabliczkach) można wyświetlać tekstury z Minecrafta i resource packów.

## Przykłady

### Chat

```mcfunction wrap
/tellraw @p ["<Nadwey> Kocham banany ",{atlas:"minecraft:blocks",sprite:"custom_emoji:emoji/heart"}]
```

![](src/assets/docs/snapshots/25w32a/chat.png)

### Title

```mcfunction wrap
/title @p title [{atlas:"minecraft:blocks",sprite:"custom_emoji:emoji/happy"},{text:" Witaj na NadweyCraft! ",color:"#2686ff",bold:true},{atlas:"minecraft:blocks",sprite:"custom_emoji:emoji/happy"}]
```

![](src/assets/docs/snapshots/25w32a/title.png)

### Lore

```mcfunction wrap
/give @p diamond[lore=["Jakość diamentu została sprawdzona,",["prez rzeczoznawcę Ryszarda",{atlas:"minecraft:blocks",sprite:"custom_emoji:emoji/happy",color:"white",italic:false}]]]
```

![](src/assets/docs/snapshots/25w32a/lore.png)

### Tabliczka

```mcfunction wrap
/setblock ~ ~1 ~ oak_sign{front_text:{messages:[[{atlas:"minecraft:blocks",sprite:"custom_emoji:emoji/heart",color:white},{atlas:"minecraft:blocks",sprite:"custom_emoji:emoji/revolving_hearts",color:white},{atlas:"minecraft:blocks",sprite:"custom_emoji:emoji/revolving_hearts",color:white},"SHGBBHS",{atlas:"minecraft:blocks",sprite:"custom_emoji:emoji/sob",color:white},{atlas:"minecraft:blocks",sprite:"custom_emoji:emoji/sob",color:white}],"",{atlas:"minecraft:blocks",sprite:"custom_emoji:emoji/absolute_cinema",color:white},""]}} replace
```

![](src/assets/docs/snapshots/25w32a/sign.png)

### Scoreboard

## Detale (technicznie)

### Komponent `object`

Jest podobny jak każdy inny [komponent tekstu](https://minecraft.wiki/w/Text_component_format), ale zamiast np. `text` i `color`,  
Używamy `atlas` i `sprite`.

- `atlas` atlasu, z którego chcemy pobrać grafikę, np. `minecraft:blocks`.
- `sprite` grafika, którą chcemy wyświetlić (zdefiniowana w atlasie).

Przykład na emotce serca z dołączonego resource packu:

```json
{atlas:"minecraft:blocks",sprite:"custom_emoji:emoji/heart"}
```

### Atlasy

Resource packi nie mogą tworzyć własnych atlasów tekstur, dlatego musimy nadpisać istniejący.  
W dołączonym resource packu, nadpisany jest atlas `minecraft:blocks`.

Nie jest to idealne rozwiązanie, inne resource packi mogą mieć problemy z kompatybilnością, a nowe wersje mogą wprowadzać zmiany w atlasach.  
Ale na czas pisania tego dokumentu, nie ma za bardzo innych opcji.

Emotki możemy dodawać do atlasu w poniższy sposób, [zresztą opisany na minecraft.wiki](https://minecraft.wiki/w/Atlas):

```jsonc title="/assets/minecraft/atlases/blocks.json"
{
  "sources": [
    // nasze emotki
    {
        "type": "minecraft:single",
        "resource": "custom_emoji:emoji/happy",
        "sprite": "custom_emoji:emoji/happy"
    },
    {
        "type": "minecraft:single",
        "resource": "custom_emoji:emoji/heart",
        "sprite": "custom_emoji:emoji/heart"
    },
    {
        "type": "minecraft:single",
        "resource": "custom_emoji:emoji/revolving_hearts",
        "sprite": "custom_emoji:emoji/revolving_hearts"
    },
    {
        "type": "minecraft:single",
        "resource": "custom_emoji:emoji/sob",
        "sprite": "custom_emoji:emoji/sob"
    },
    {
        "type": "minecraft:single",
        "resource": "custom_emoji:emoji/flushed",
        "sprite": "custom_emoji:emoji/flushed"
    },
    {
        "type": "minecraft:single",
        "resource": "custom_emoji:emoji/absolute_cinema",
        "sprite": "custom_emoji:emoji/absolute_cinema"
    },

    // obiekty minecrafta (reszta pliku jest taka sama jak atlas `blocks.json` w plikach minecrafta)
    {
      "type": "minecraft:directory",
      "prefix": "block/",
      "source": "block",
    },
    // ...
  ],
}
```
